-----------------------English-----------------------
To run the website, please follow these instruction:
1. Create a database name "do_an" in your phpmyadmin page.
2. Enter "localhost/(our project folder)" or "localhost:8888/(our project folder)" to the address bar on your browser
3. Our website will begin at the register screen to make your own account, if you already have account, please click on the login text below.
4. In home page, you can check any requirement features in our website, any extra features are being developed.
